# Gabriel Etchegoyen — Portfolio Personal

Portfolio personal desarrollado con HTML, CSS y JavaScript vanilla. Diseño dark-mode con estética de terminal, orientado a mostrar mi perfil como Backend Developer y Analista Funcional.

🌐 **Demo en vivo:** [gabyetche-portfolio.netlify.app](https://gabyetche-portfolio.netlify.app)

---

## Stack

- HTML5
- CSS3 (sin frameworks, media queries custom)
- JavaScript vanilla
- Google Fonts: JetBrains Mono + Space Grotesk

## Features

- Terminal animada con efecto typing en el hero
- Navegación con scroll spy (link activo por sección)
- Menú hamburger para mobile (≤767px)
- Carrusel de imágenes en la galería de proyectos
- Breakpoints responsive: 375px / 425px / 767px / 768px / 930px / 1010px / 1130px
- Botón Back to Top
- Noise overlay decorativo
- Descarga directa de CV

---

## Secciones

- **Hero** — presentación con terminal interactiva
- **Sobre mí** — perfil profesional y stats
- **Skills** — tecnologías con nivel de experiencia
- **Proyectos** — cards con links a repos y demos
- **Experiencia** — trayectoria laboral
- **Formación** — cursos y certificaciones
- **Contacto** — links a redes y descarga de CV
- **Galería** — página aparte con detalle de cada proyecto
